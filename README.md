# Citadle

## Introduction

This is the frontend for Citadle, a Wordle like game for guessing the names of cities.

## Setup

To get started, you will need the following in your .env file:

```bash
VITE_NODE_ENV=
VITE_API_URL=
```

## Development

To start the development server run:

```bash
bun run dev
```
